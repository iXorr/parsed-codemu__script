## DESCRIPTION

This is an experimental project created as part of a small team work.

A PHP script for web servers that parses external page and modifies content using a custom word-replacement dictionary.

In my case, it's code.mu - web-education resource I had been using. My classmates replaced titles of web topics on "emotional" words in ``corrects.php`` - injector's vocabulary.
